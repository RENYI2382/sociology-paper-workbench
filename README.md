# 社会学论文工作台

面向社会学、社会工作与一般社会科学论文写作的个人 skill 项目。

这个项目不是通用型写作助手，而是把多年积累下来的论文 prompt、文献阅读模板、综述写法、降 AI 味经验和审稿式检查方法整理成一个可以持续维护的 skill 工作台。

## 为什么做这个项目

做社会学论文时，最耗时间的往往不是真正的研究本身，而是反复处理这些高频环节：

- 文献检索后如何快速进入阅读
- 读完之后如何提炼成可引用、可进入综述的材料
- 国内外研究现状怎么避免写成“作者 + 年份 + 观点”的流水账
- 正文怎么写得像社会科学论文，而不是像模板拼接
- 稿子完成后，怎么再做一轮降 AI 味和严格自查

过去几年里，这些环节我分别积累了很多 prompt，但它们原本分散在不同文件里，命名方式也偏阶段性记录。这个项目的目的，就是把这些零散积累整理成一套真正可复用的社会学论文工作流。

## 我们做了什么

这次整理主要完成了三件事：

- 保留原始 prompt 内容，不改写、不删改原文
- 按论文工作流重新分类，而不是继续按时间或零散文件名堆放
- 为中文期刊版和英文期刊版分别建立可直接使用的入口说明

目前，这个 skill 已经覆盖从文献检索、基础信息提取、深度阅读、文献综述、正文改写、降 AI 味到审稿诊断的完整链条。

## 特点

- 原件保留：所有原始 prompt 已原样归档，方便追溯和继续积累
- 工作流化：不是单个 prompt 集合，而是一套论文写作流程
- 学科取向明确：强调社会学和社会工作的机制分析写法
- 双版本支持：同时面向中文期刊版和英文期刊版
- 易于扩展：后续可以继续按模块追加新的 prompt 资产

## 安装方式

本项目当前以 Trae 本地 skill 的形式使用，不需要单独发布到外部仓库平台。

只要将 skill 文件夹保存在当前工作区的 `.trae/skills/` 目录下即可被识别：

```text
.trae/skills/sociology-paper-workbench/
```

如果你后续要迁移到新的工作区，只需复制整个目录：

```text
sociology-paper-workbench/
```

并放到目标项目的：

```text
.trae/skills/
```

目录中即可。

## 快速开始

如果你已经把本 skill 放在 `.trae/skills/` 下，可以直接开始按任务调用：

1. 先判断当前任务属于中文期刊版还是英文期刊版
2. 再判断当前阶段属于检索、阅读、综述、改写、降 AI 味还是审稿
3. 打开对应的工作流说明文件
4. 再去调用相应目录里的原始 prompt

推荐先看：

- `SKILL.md`
- `CN_JOURNAL_WORKFLOW.md`
- `EN_JOURNAL_WORKFLOW.md`
- `PROMPT_MAP.md`

# Part I: 写作 Prompt 集合

## 文献检索与资料池

这一部分主要用于搭建研究初期的资料池，帮助快速锁定主题、收集 DOI、形成检索入口，并为后续阅读与综述写作打基础。

对应材料包括：

- `1.1_essay download search_prompt.rtf`
- `1.2_essaydown_healthinequality_dofile.txt`
- `literature review SOP.txt`

## 文献基础信息提取

这一部分用于从题目、摘要、网页、截图或 PDF 文本中快速提取标准字段，适合做文献台账、阅读卡片和资料整理。

对应材料包括：

- `0915prompt_飞书文献阅读_基本信息.rtf`
- `251128文献综述1_基本信息的提取.rtf`

## 深度阅读与结构化笔记

这一部分用于把单篇文献转化成可以直接服务论文写作的阅读笔记，重点不是摘要复述，而是重建文章的逻辑主线、研究问题与研究价值。

对应材料包括：

- `2.1_prompt_wenxian1_学术阅读人格设定.rtf`
- `2.2_prompt_wenxian2_学术文献阅读总结.rtf`
- `2511128文献综述2_阅读总结.rtf`
- `251128文献综述助手_三合一版.rtf`
- `251213文献阅读助手v5.0.rtf`
- `251213文献阅读助手v5.0_English version.rtf`
- `251128文献综述3_文献分析.rtf`

## 文献综述与研究现状写作

这一部分主要解决“文献怎么写成对话”这个问题。重点不是堆叠前人观点，而是形成回顾、比较、评论与 gap 提炼的连续结构。

对应材料包括：

- `3.1_文献综述优化prompt1_模板编写.rtf`
- `3.2_文献综述优化prompt2_文献对话.rtf`

## 社会学与社会工作正文写作

这一部分面向真正的社科论文表达，强调条件、机制、后果和论证推进，避免情绪化、空泛化和“换个视角说一遍”的写法。

对应材料包括：

- `社工论文编写要求（260106）.txt`
- `通用社科类学术论文写作助手 Prompt.txt`

## 降 AI 味与模板味改写

这一部分主要处理论文成稿后的风格细修，目标不是简单替换词语，而是在保留学术感的前提下，降低模板味、翻译腔和泛化表达。

对应材料包括：

- `降低AI味道规范_260328_.rtf`
- `降低AICG提示词2.0_260331.txt`

## 严格审稿人式检查

这一部分用于投稿前的严格自查。它更接近真正的同行评审思路，会优先检查原创性、研究问题、方法、证据和论证链条，而不是只修语言表面问题。

对应材料包括：

- `严肃的审稿人_prompt.rtf`

## 学位论文与章节构筑

这一部分偏向更大体量的论文结构设计，适合用来搭建研究开篇、研究设计、结果呈现与章节逻辑。

对应材料包括：

- `博士论文的构筑_prompt.rtf`

# Part II: 论文写作相关的 Skills

## Skills 的结构

当前 skill 项目采用“原件保留 + 索引说明”的方式组织：

- `SKILL.md`：供 Trae 调用的技能说明文件
- `source-prompts/shared/`：中英文通用的原始 prompt
- `source-prompts/chinese-journals/`：偏中文期刊版的原始 prompt
- `source-prompts/english-journals/`：偏英文期刊版的原始 prompt
- `CN_JOURNAL_WORKFLOW.md`：中文期刊版工作流入口
- `EN_JOURNAL_WORKFLOW.md`：英文期刊版工作流入口
- `PROMPT_MAP.md`：原始材料与功能模块的映射表

## 中文期刊版

中文期刊版适合以下任务：

- 中文社会学期刊论文
- 中文社会工作期刊论文
- 中文文献综述
- 中文学位论文章节

这一版更强调：

- 机制解释
- 段落推进
- 小标题简洁严谨
- 理论内嵌式表达
- 降低模板味和空泛表态

## 英文期刊版

英文期刊版适合以下任务：

- 英文期刊投稿准备
- 英文摘要与英文综述
- 英文论文结构整理
- reviewer 式英文稿件自查

这一版更强调：

- 问题、方法、结论链条清楚
- research gap 明确
- 表达清晰、证据导向
- 投稿前按 reviewer 标准提前检查

## 使用方式

推荐的使用顺序如下：

1. 先确定任务属于中文期刊版还是英文期刊版
2. 再判断当前处在检索、阅读、综述、改写、降 AI 味还是审稿阶段
3. 按工作流入口去调用对应目录下保留的原始 prompt
4. 在后续扩展时继续把新 prompt 归入对应模块，而不是零散保存

## 维护说明

为了保证这个 skill 能长期使用，建议按下面的方式维护：

### 1. 保留原件，不覆盖旧版本

新增 prompt 时，优先保留原文件，不要直接覆盖旧材料。这样能保留你不同阶段的写法轨迹，也方便以后回看哪些版本更适合自己。

### 2. 先归类，再扩写说明

如果你新增了一个 prompt，建议优先做两步：

1. 放入 `source-prompts/` 对应目录
2. 在 `PROMPT_MAP.md` 中补上映射关系

只有在确实形成稳定使用场景时，再去补充工作流说明。

### 3. 中文版和英文版分开维护

中文期刊版和英文期刊版虽然可以共享一部分材料，但最好继续分别积累：

- 中文版重点补机制分析、综述对话、降模板味
- 英文版重点补英文润色、英文综述写法、reply to reviewer

### 4. README 作为项目首页

后续这个项目如果继续扩展，`README.md` 最好只承担四个功能：

- 说明项目定位
- 提供目录跳转
- 展示安装与使用方式
- 指向更细的工作流文件

不要把所有细节都塞进 README，具体说明应下沉到工作流文件和映射表中。

## 后续扩展

这个项目下一步最值得补充的，不是再写新的概念说明，而是继续积累两类高价值材料：

- 真实论文片段的修改前后对照
- 你所在研究方向常用理论与经验分析的写法模板

这样，这个 skill 才会从“整理好的 prompt 集”进一步升级为“真正有学科风格的个人写作系统”。
