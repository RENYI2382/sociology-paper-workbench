# 社会学与社会工作论文全流程工作台 
(Sociology Paper Workbench)

这是一个专为**社会学**、**社会工作**及**社会科学**硕士/博士研究生打造的论文写作 Skill 工作台。

这不是一个简单的“润色工具”，而是将数年来积累的优质 Prompt、文献阅读模板、机制写作法、降 AI 味经验和审稿诊断策略，沉淀为一套可复用的工程化工作流。

---

## ⏱️ 30 秒看懂

| 你给它什么 | 它帮你做什么 |
| :--- | :--- |
| 一篇核心文献 (PDF/文本) | 📚 **深度解构阅读**：不只是摘要，而是提炼研究问题、核心机制与理论对话点 |
| 几篇相关文献/阅读卡片 | 🔄 **文献综述与学术对话**：拒绝“作者+年份”流水账，生成具有批判性和 Gap 意识的学术对话 |
| 一篇“机翻感”初稿 | 🛠️ **降 AI 味与风格精修**：消除模板感、排比句和泛泛而谈，恢复社科特有的严谨、克制口吻 |
| 准备投稿的全文 | 🧐 **严厉的审稿人诊断**：从逻辑链条、证据强度、理论贡献进行“毒舌”预审与拒稿风险排查 |
| 一个模糊的研究主题 | 🔍 **文献检索与资料池**：快速构建中英文检索式，建立文献台账基础 |

**核心判断：**
可用作正文 / 需补充机制证据 / 仅作文献参考 / 逻辑断裂需重构

**核心产出工作流：**
`原始文献 -> 深度阅读卡片 -> 批判性文献综述 -> 社科风格正文 -> 降AI味精修 -> 审稿人意见表`

```text

    └── 06_phd_thesis_...    # 博士/硕士学位论文大纲构筑
```

## 📂 核心 Prompt 目录与功能说明

我们在不改变原始优质逻辑的基础上，对历史积累的 Prompt 进行了规范化重命名：

### 1. 深度阅读 (Deep Reading)
> 经过精简，仅保留最强大、最完善的 v5.0 版本，提供中英双语支持。
- `02_deep_reading_cn.rtf`：**中文文献阅读助手 v5.0**。结构化拆解中文文献的研究背景、核心机制、数据方法与结论。
- `02_deep_reading_en.rtf`：**英文文献阅读助手 v5.0**。专为英文顶级期刊设计，精准捕捉理论贡献与 Research Gap。

### 2. 文献综述 (Literature Review)
- `03_lit_review_template.rtf`：**文献综述模板编写**。构建文献综述的骨架，避免记流水账。
- `03_lit_review_dialogue.rtf`：**文献对话生成器**。将不同文献进行对比、批评与整合，写出真正的“学术对话”。
- `03_lit_review_sop.txt`：**综述标准化操作流程**。

### 3. 降 AI 味与精修 (AI Trace Reduction)
- `04_ai_trace_reduction_v1.rtf`：**降 AI 味道规范 (基础版)**。消除常见的“首先、其次、最后”、“综上所述”等机器感词汇。
- `04_ai_trace_reduction_v2.txt`：**降 AI 提示词 2.0 (进阶版)**。深度重构长难句，增强学术克制感，强化机制解释。

### 4. 审稿与构筑 (Review & Structuring)
- `05_reviewer_criticism.rtf`：**严肃的审稿人**。扮演挑剔的 CSSCI/SSCI 匿名审稿人，对你的稿件进行无情批判并给出修改建议。
- `06_phd_thesis_structuring.rtf`：**博士论文构筑**。用于规划大体量学位论文的章节逻辑与宏观理论框架。

### 5. 基础规范与检索 (Guidelines & Search)
- `00_guidelines_social_work.txt` / `00_guidelines_general_social_science.txt`：全局遵守的社科/社工底层写作要求。
- `01_literature_search.rtf`：文献检索式生成与初步筛选。

## 💡 直观的 Prompt 展示与输出示例

为了让你更好理解这些 Prompt 的威力，以下是几个核心模块的片段展示与实际输出效果：

### 🌟 示例 1：深度阅读卡片生成
**调用的 Prompt 核心片段 (`02_deep_reading_cn.rtf`)：**
> "你现在是一位严谨的社会学博士后，请仔细阅读我提供的文献。不要仅仅做摘要复述，请必须明确提炼出：1. 核心研究问题是什么？ 2. 核心机制（X如何影响Y）是什么？ 3. 作者与谁进行了理论对话？ 4. 本文的局限性在哪？请用学术化、凝练的语言输出表格和要点..."

**最终输出示例：**
```markdown
### 核心机制拆解
- **自变量 (X)**: 互联网使用频率
- **因变量 (Y)**: 阶层认同感
- **中介机制**: 互联网使用并未直接提升经济地位，而是通过“扩大参照群体”和“增加相对剥夺感”，反而降低了主观阶层认同。
- **理论对话**: 本文反驳了技术乐观主义中的“信息平权假说”，转而支持了“相对剥夺理论”。
```

### 🌟 示例 2：文献综述“对话式”改写
**调用的 Prompt 核心片段 (`03_lit_review_dialogue.rtf`)：**
> "我将提供几篇文献的笔记。请不要使用'张三认为...李四认为...'的句式。请找出它们在观点上的分歧、补充或递进关系。按主题维度（而非作者维度）组织段落，用'然而'、'与此相对'、'进一步地'等逻辑词串联，写出具有批判性的文献综述段落。"

**最终输出示例：**
```markdown
早期的社会支持理论主要关注网络规模的扩大对心理健康的积极作用（Smith, 2018）。然而，近期的研究开始注意到网络结构的异质性。与 Smith 强调“量”的逻辑不同，Wang 等人（2021）指出，强连带的过度密集反而会形成信息茧房，导致情感耗竭。进一步地，赵某某（2022）在本土语境中证实，这种情感耗竭在缺乏体制内保障的流动人口中尤为显著。因此，现有文献在“网络效价”上存在分歧，而本研究正是试图引入“制度环境”这一调节变量来弥合该鸿沟。
```

### 🌟 示例 3：无情降 AI 味
**调用的 Prompt 核心片段 (`04_ai_trace_reduction_v2.txt`)：**
> "重写以下文本。禁止使用排比句；禁止使用'不仅...而且...更'；禁止出现'具有重要意义'等空泛表态。将主谓宾结构缩紧，多使用学术名词替代描述性短语，保留事实、数据和机制论证，删除所有抒情和过度总结。"

**修改前 (AI 味浓厚)：**
> 综上所述，社会工作介入不仅能够有效地帮助老人缓解孤独感，而且还能显著地提升他们的社区参与度，这对于构建和谐社会具有极其重要的深远意义。

**修改后 (学术严谨)：**
> 研究表明，社会工作介入在缓解老年群体孤独感的同时，有效促进了其社区参与。这一干预路径为基层社区治理与社会整合提供了实证依据。
